the next javascript library

What is JAction?

JAction is a Javascript library created to make it easier for users to manipulate the DOM and Ajax

JAction has the advantage that it can run in all browsers (including IE9) and is suitable for dynamic website development



Why JAction?

This library is written originally in the javascript programming language, so it does not require certain conditions, this library is also suitable for developing applications based on Electron



What are the conditions for using JAction

You will need to master some basic javascript programming, such as ES6

Example

An example program that uses JAction to add button events

JAction begins by declaring a JAction object or using the 'jct' character as an alias to speed up writing

      
//with alias 'jct'
jct("button").event ("click", function (){
   alert ("yes")
 }) 



//without alias 
JAction("button").event ("click", function (){
   alert ("yes")
})
